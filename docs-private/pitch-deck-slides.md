# Le paradoxe de l'investissement

## Des décisions à millions, des outils à zéro

- **Business Angel** : décisions à 25-50K€ — analyse au feeling
- **Fonds minoritaire** : 1 analyste junior débordé pour 200 deals/an
- **PE / M&A** : cabinets à 200-500K€/an pour des processus longs et rigides
- **PitchBook, CB Insights, Dealroom** : fournisseurs de données, pas partenaires de décision
- **ChatGPT seul** : réponses enthousiastes, sans contradiction, parfois avec des chiffres inventés

> Ce qui manque n'est pas un outil de plus. C'est un **partenaire de décision**.

# Comment ça marche

## 4 tiers d'analyse orchestrés

- **Tier 0 — Extraction** : chaque donnée classifiée par fiabilité (6 niveaux : AUDITED → UNVERIFIABLE)
- **Tier 1 — Analyse** : 13 agents en parallèle (audit financier, équipe, marché, concurrence, juridique, tech, sortie...)
- **Tier 2 — Expert sectoriel** : 1 expert parmi 22 spécialités évalue le deal vs les benchmarks de son industrie
- **Tier 3 — Synthèse** : contradictions entre agents, scoring, devil's advocate, scénarios, mémo d'investissement

> Aucun analyste humain n'est expert dans 22 secteurs simultanément. Angel Desk si.

---

# La sublimation

## L'IA qui se transcende en se confrontant

**Board AI** : 4 modèles (Claude, GPT-4o, Gemini, Grok) en débat structuré

- Chaque modèle voit les positions des autres
- Peut changer d'avis s'il est convaincu par les preuves
- Doit défendre ses divergences avec des données

**Résultat** :
- Les biais se neutralisent
- Les hallucinations se font attraper
- Le consensus est structurellement plus fiable qu'un avis unique

> "Et si l'IA se trompe ?" — C'est exactement pour ça qu'on a construit la délibération multi-modèle.

# Le Live Coaching

## Un partenaire temps réel pendant les calls fondateurs

Pendant l'appel, l'investisseur reçoit en **< 8 secondes** :

- **Contradictions** : ce que le fondateur dit vs ce que le deck affirme
- **Benchmarks** : un chiffre annoncé vs la réalité du marché
- **Questions critiques** : à poser maintenant, pas après
- **Nouvelles infos** : données qui changent le score d'un agent

Pipeline : transcription audio + analyse visuelle des slides + contexte complet de la DD

Après l'appel : rapport post-call + re-analyse automatique des agents impactés
